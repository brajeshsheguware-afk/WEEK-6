<h1 align="center"> CUSTOMER CHURN ANALYSIS SYSTEM</h1>

## OVERVIEW

This project analyzes customer churn using **synthetic data** to identify patterns and factors that lead to customer loss. It uses Exploratory Data Analysis (EDA) to understand customer behavior and improve retention strategies.

---

## OBJECTIVE

* Generate synthetic customer dataset
* Perform data cleaning and preprocessing
* Analyze churn behavior using EDA
* Identify key factors affecting churn

---

## TECHNOLOGIES USED

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## DATASET DETAILS

The dataset contains **100,000 customer records** with features such as:

* Customer_ID
* Age (18–70)
* Gender
* Tenure (0–10 years)
* Balance (0–200,000)
* Credit Score (300–900)
* Estimated Salary
* Number of Products (1–4)
* Active Member Status
* Churn (0 = No, 1 = Yes)

---

## FEATURES & WORKFLOW

### 1. DATA GENERATION

* Synthetic dataset created using NumPy
* Simulates real-world customer scenarios

### 2. DATA CLEANING

* Checked missing values
* Removed duplicates
* Encoded categorical variables

### 3. STATISTICAL ANALYSIS

* Mean, Median, Mode
* Standard Deviation
* Distribution analysis

### 4. CHURN ANALYSIS

* Segmented customers into churn and non-churn
* Identified high-risk groups

### 5. GROUP ANALYSIS

* Age group vs churn
* Gender vs churn
* Active vs inactive users
* Product usage vs churn

### 6. RELATIONSHIP ANALYSIS

* Balance vs churn
* Credit score vs churn
* Tenure vs churn

## KEY INSIGHTS

* Low balance and inactive customers have higher churn
* Customers with low tenure are more likely to leave
* Fewer products increase churn probability
* Active users show lower churn rates
